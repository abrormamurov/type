Masalan-1

Type conversion - Javascript dasturlash tilida saqlanayotgan ma'lumot turini bir ko'rinishdan ikkinchi ko'rinishga o'tkazish

Number - primitiv o'rab turuvchi obyektidan foydalanib, berilga malumot turini son ko'rinishga o'tkazish imkonyati mavjud. Agar berilgan ma'lumot son ko'rinishiga o'tkazishni imkoni bo'lmasa, NaN - (Not a Number) maxsus qiymati yuzaga keladi.

const yosh = '23'

// String

console.log(yosh);

// String

console.log(Number(yosh));

// Number


console.log(yosh+1);

// 231 (concatenation)

console.log(Number(yosh) + 1);

// 24


const ism = 'Abrorbek';
console.log(Number(ism));

// NaN


String

String - primitiv o'rab turuvchi obyektidan foydalanib, berilgan ma'lumot turini string ko'rinishga o'tkazish imkoniyati mavjud.


const yil = 2024

// Number

console.log(yil);

// Number

console.log(String(yil));

// String

Masalan-2

Type coercion - Javascript dasturlash tilida operator turli xil qiymatlar ustida ish olib borayotganda, ulardan birinchi avtomatik ravishda boshqa turga o'tish. Boshqa qilib aytganda ma'lumot turini avtomatik bir ko'rinishdan ikkinchi korinishga o'tish.

Auto string conversion

console.log('Men' + 2005 + 'chi yilda tavallud topganman');

// auto string con

Auto number conversion

console.log('30' - '10' - 5);

// auto number conversion
